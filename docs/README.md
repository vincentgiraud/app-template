# Documentation

This folder is populated by the agent pipeline.

## Generated Documents

| Document | Created by | When |
|----------|-----------|------|
| `stack-report.md` | az-saas-planner | During Azure stack planning — consolidates compliance, security, cost, and growth assessments |

Architecture, frontend, backend, and task specs are written to `.github/instructions/` where Copilot loads them automatically.

## Getting Started

1. Run `@Spec Planner` with your project idea
2. Specs are generated in `.github/instructions/`, stack report here in `docs/`
3. The `docs-updater` cloud agent keeps docs current as code changes
